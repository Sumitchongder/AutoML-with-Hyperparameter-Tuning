1. Create a new branch:

   ```bash

   ```

2. Commit changes with **clear and descriptive messages**

3. Push to your fork and open a Pull Request (PR)

   * Motivation and impact

### Pull requests are reviewed for:

* Code quality
* Correctness
* Reproducibility
* Alignment with project goals
